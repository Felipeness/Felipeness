## Building in public

**[nessy](https://github.com/Felipeness/nessy)** — Nano Episodic Session Studio. Local indexer + analytics for Claude Code sessions. Hybrid search (BM25 + embeddings + RRF) over `~/.claude/projects/*.jsonl`. Go + SQLite + Ollama (optional). Single binary, zero cloud egress.

**[The Whole and the Part](https://github.com/Felipeness/the-whole-and-the-part)** — Open-source book on Holonomic Software Architecture. Koestler's holons applied to distributed systems. 62 chapters. CC BY-SA 4.0.

**[appointment-api](https://github.com/Felipeness/appointment-api)** — Healthcare scheduling reference. NestJS + Prisma + SQS. Transactional Outbox + Saga + Circuit Breaker. Demo only.

**[holonomic-architecture](https://github.com/Felipeness/holonomic-architecture)** — Production reference for Holonomic Systems. Fastify + Effect-TS + Temporal + schema-per-holon. Law of Imports enforced.

**[claude-statusline](https://github.com/Felipeness/claude-statusline)** — Custom statusline for Claude Code with web visual editor. Single Go binary. 16 components, 5 themes, drag-and-drop.
